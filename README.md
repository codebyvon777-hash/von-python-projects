# von-python-projects
My Python projects made on Galaxy A42 5G

## Modern Calculator App
Built with Kivy for Android (Pydroid 3)

### Features

#### 1. Standard Calculator
- Add, subtract, multiply, divide
- Percent (%) and parentheses support
- Clear (AC) and backspace (X) buttons
- Live preview of calculation results as you type

#### 2. Profit Mode
- Calculate selling price and profit based on:
  - Cost
  - ROI (Return on Investment) %
  - Shipping or other costs
- Displays: Profit and Selling Price

#### 3. Wallpaper Customization
- Choose a custom image as the calculator background
- Path saved so wallpaper persists between sessions

#### 4. Secret Commands
- `BTC` → shows current Bitcoin price in USD
- `ETH` → shows current Ethereum price in USD
- `HASH <text>` → computes SHA-256 hash of the text
- `ROI <p r t>` → computes profit using formula `p*(1+r/100*t)`
- `NOTE` → opens secret notes editor where you can write and save text

### Usage
- Tap buttons for numbers and operators
- Switch between "Calculator" and "Profit Mode" using top buttons
- Tap "Wallpaper" to change the background
- Enter secret commands in the calculator input for advanced functions
